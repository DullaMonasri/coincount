# TravelWise

Personal Expense & Travel Budget Tracker

## Overview

TravelWise is a Flask-based web application that helps users manage:

- Personal Expenses
- Income
- Budgets
- Savings Goals
- Travel Planning
- Travel Expenses
- Reports & Analytics

## Features

### User Authentication

- Register
- Login
- Logout
- Profile

### Expense Management

- Add Expense
- View Expense
- Delete Expense
- Search Expense

### Income Management

- Add Income
- View Income
- Delete Income

### Budget Management

- Set Budget
- Track Spending

### Savings Goals

- Create Goals
- Track Savings

### Travel Management

- Create Trips
- Manage Travel Budgets

### Reports

- Expense Reports
- Budget Reports
- Travel Reports

### Charts

- Pie Chart
- Bar Chart
- Line Chart

## Installation

Install Dependencies

```bash
pip install -r requirements.txt
```

Create Database

```bash
sqlite3 database.db < schema.sql
```

Run Application

```bash
python app.py
```

Open Browser

```text
http://127.0.0.1:5000
```
